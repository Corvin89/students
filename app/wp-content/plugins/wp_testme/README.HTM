Установка

  1. Распакуйте папку с плагином в папку wp-content/plugins/
  2. В Панеле Администратора выберите вкладку "Плагины" и нажмите "Установить" рядом с плагином TESTME - Плагин для создания тестов.

Обновление

  1. Если вы не вносили изменения в файл стилей плагина, то загрузите все файлы в папку с плагином.
  2. Если вы вносили изменения в файл стилей плагина, то замените все файлы, кроме testme_style.css.
  3. В Панеле Администратора выберите вкладку "Плагины" и нажмите "Обновить" рядом с плагином TESTME - Плагин для создания тестов.
  4. Если у вас были проблемы с установкой прошлой версии, то деактивируйте плагин и активируйте заново.

Создание теста

  1. В разделе меню "TESTME" выберите вкладку "Тесты".
  2. Нажмите на ссылку "Добавить тест", введите название теста и нажмите на кнопку "Добавить тест". Новый тест появится в списке тестов.
  3. Нажмите на иконку "Редактировать" рядом с названием теста.
  4. Шаг 1: Заполните данные теста. Сохраните.

    Шаг 2: Введите вопросы и варианты ответов на них. Если тест типа Абв, то в зеленых полях рядом с ответом недо ввести букву (или несколько букв через запятую), которые соответствуют этому ответу. Подходят любые буквы (русские, латинские) и цифры. Сохраните.

    Шаг 3: Введите результаты. Для результата надо ввести как минимум 1 из трех полей: "Заголовок результата", "Картинка", "Описание". Можно все три. Сохраните.

    Шаг 4: Проверьте данные теста. Если тест готов, нажмите на кнопку "Тест готов". Автоматически будет сформирован черновик с кодом теста.

  5. Перейдите в раздел "Записи", отредактируйте черновик по своему усмотрению и опубликуйте запись.

Добавление теста в запись блога

  1. Начиная с версии 1.3 записи формируются автоматически. Н это можно сделать и вручную, добавив в запись блога код [TESTME N], где N - номер теста.

Всякое разное

  1. Текст, который опубликован в записи блога до кода [TESTME N] будет показываться и до вопросов, и до результатов. Текст, который написан в поле "Описание теста" публикуется только перед списком вопросов.
  2. В одном тесте может быть разное количество вариантов ответов к разным вопросам.
  3. Для изменения дизайна вопросов, ответов и результатов надо внести изменения в файл стилей testme_style.css .
  4. Загруженные картинки могут не вставляться в результаты, если поле "Ссылка" пустое. Нажмите на надписи "Ссылка на файл" или "Ссылка на запись", чтобы это поле заполнилось.
  5. Загруженные картинки из результатов и описания теста могут автоматичсеки прикрепиться к созданной записи при условии, что они вставлены в полный размер. Если тема поддерживает миниатюры, то соотвествующая картинка может быть атвоматически взята из описания теста. Картинки, котрые вставляются в уменьшенном размере, автоматически не распознаются как привязанные.

История плагина

Версия 1.3 от 14.06.2011

Спонсор этой версии - http://trikky.ru.

  • Плагин приведен в соответствие с версией Wordpress 3.1 и Php 5.3.
  • Добавлена возможность отключить статистику.
  • Создание теста разбито на шаги.
  • Появилась возможность менять тип теста.
  • Появилась возможность добавлять сколько угодно вопросов, ответов и результатов в любое время.
  • Появилась возможность загружать картинки напрямую в результаты.
  • Автоматическое создание записей с кодом теста.
  • Появилась возможность выводить вопросы и ответы в случайном порядке.

Версия 1.2 от 06.10.2009

Спонсор этой версии - Михаил.

  • Добавлена возможность разрешить проходить тест только зарегистрированным пользователям
  • Добавлена возможность добавить надпись "Вы набрали N баллов из X возможных.
  • В статистике добавлена таблица со списком последних пользователей, прошедших тест с указанием точного числа набранных баллов.

Версия 1.1 от августа 2009

  • Позволяет создавать тесты двух видов (abc и 123), изменять и публиковать их.
  • Показывает статистику тестов: сколько человек получило каждый из результатов.
